# Greener Frontend

Greener is a web application to promote sustainability by tracking waste, joining eco-challenges, and engaging with a community. This repository contains the frontend, built with React.

## Project Structure
- **src/**: Source code, including components, services, and styles.
- **public/**: Static assets like icons and manifest.
- **docs/**: Comprehensive documentation (see below).
- **node_modules/**: Dependencies.

## Documentation
Find detailed guides in the `docs/` directory:
- **Research**: Technology stack, architecture, design decisions (`docs/research/`).
- **User Manual**: Guides for end-users (`docs/user-manual/`).
- **System Manual**: Developer setup and technical details (`docs/system-manual/`).
- **Testing**: Testing instructions and examples (`docs/testing/`).

## Setup
1. Clone the repo: `git clone <url>`
2. Install dependencies: `npm install`
3. Start the app: `npm start` or `npm install react-router-dom @reduxjs/toolkit react-redux axios`
4. See `docs/system-manual/setup-guide.md` for details.

## Running Tests
Run `npm test` to execute Jest tests. See `docs/testing/` for more.

## Contributing
- Follow the setup guide in `docs/system-manual/`.
- Add components to `src/components/` and tests alongside them.
- Update documentation in `docs/` for new features.
